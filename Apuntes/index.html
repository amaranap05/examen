html

Definición:

HTML, que significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje estándar para crear y estructurar el contenido de una página web. HTML es un lenguaje de marcado, lo que significa que usa etiquetas (o "tags") para definir y organizar distintos elementos en un documento, como encabezados, párrafos, enlaces, imágenes, tablas, formularios y más.

Estructura básica

Estructura de Documento: HTML organiza el contenido en una estructura jerárquica. Comienza con una etiqueta html, que contiene la estructura básica de un documento HTML: head: Incluye información sobre el documento, como el título (title), enlaces a archivos CSS, scripts de JavaScript y metadatos. body: Contiene el contenido visible de la página web, como texto, imágenes, enlaces, y otros elementos.

CSS

Definición

CSS (Cascading Style Sheets) es un lenguaje de diseño que se utiliza para controlar la presentación visual de documentos HTML o XML en la web. En otras palabras, CSS es lo que permite dar estilo y formato a las páginas web, haciendo que los sitios se vean atractivos y sean fáciles de usar.

JavaScript

Definición

JavaScript es un lenguaje de programación que se utiliza principalmente para agregar interactividad y dinamismo a las páginas web. Es uno de los tres pilares del desarrollo web, junto con HTML y CSS
